# FETİH Daemon + Web UI Geliştirme

FETİH projesine daemon modu ve web UI iyileştirmesi yapıyoruz.
Çalışma dizini: /home/ara/Desktop/fetih-github/fetih-main

ÖNEMLİ: LÜTFEN ADIM ADIM ÇALIŞ. Her adımda:
1. Ne yapacağını kısaca belirt
2. Kodu yaz/uygula
3. Çalıştığını doğrula
4. Bir sonraki adıma geç

Önce SPEC-DAEMON-WEB.md dosyasını oku. Orada tüm detaylar var.

ADIM SIRASI:
1. src/daemon.ts - Daemon manager (startDaemon, stopDaemon, getDaemonStatus)
2. src/daemon-log.ts - Logger
3. src/cli.ts - CLI flag güncellemesi (fetih daemon start/stop/status, --daemon)
4. src/web/server.ts - REST API endpointleri (/api/chat, /api/status, /api/sessions)
5. Web UI Vite kurulumu - web/vite.config.js
6. Web UI componentleri modülerleştir
7. Build scriptleri ve package.json güncellemesi
8. Testleri çalıştır (npm test)
9. Son kontroller
10. Tamamlandığında: touch /tmp/fetih-daemon-done

Her adımda yaptığını ekrana yaz (--print modu).
